# 🛒 Amazon Clone - Frontend 

This is a high-fidelity frontend replica of the Amazon homepage, created to demonstrate advanced CSS layout techniques and responsive web design.

## 🚀 Live Demo
[View the Live Project here](https://nikitawani-dev.github.io/amazon-clone/)

## 🛠️ Features
- **Header & Nav:** Functional search bar UI and account/orders dropdown stubs.
- **Hero Section:** Large background image with a fading effect to match the original site.
- **Product Cards:** Responsive grid layout that adjusts based on screen size.
- **Footer:** Multi-level footer with "Back to Top" functionality.

## 📁 Project Structure
- `index.html`: The main structure of the page (renamed from amazon.html).
- `style.css`: All custom styling, including Flexbox and Grid.
- `*.avif / *.jpg`: Optimized product and hero images for fast loading.

## 💻 Tech Used
- HTML5 (Semantic Tags)
- CSS3 (Flexbox, Grid, Media Queries)
